# Data Directory

This folder contains downloaded rental data from official Irish government sources.

## Data Sources

### RTB Rent Index (via CSO)
- **Table ID:** RIQ02
- **Source:** Residential Tenancies Board / Central Statistics Office
- **URL:** https://data.cso.ie/
- **Frequency:** Quarterly
- **Coverage:** 2007 onwards
